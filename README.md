# Immerse

A highly 3D-designed website built as a personal exploration of WebGL, scroll-driven motion, and modern Next.js. Curiosity project — conceived one morning, built across an afternoon.

## Stack

- Next.js 16 + React 19 + TypeScript
- Three.js (3D scenes)
- GSAP + Framer Motion (motion + scroll-driven sequences)
- Tailwind CSS 4 + shadcn/ui

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
